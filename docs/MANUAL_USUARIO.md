# Manual de Usuario - AutoReplyPro

## Acceso
- Frontend: http://localhost:3000
- Backend: http://localhost:8000

## Calendario
- Vista mensual con calendario central.
- Click en un dia: abre pop up con agenda del dia y formulario para crear turno.
- Boton "Crear turno": abre el mismo pop up.

## Crear turno
1) Escribe el nombre completo (con sugerencias).
2) Selecciona tratamiento.
3) Selecciona fecha.
4) Selecciona hora disponible.
5) Click en "Crear".

## Pacientes
- Lista de pacientes con busqueda.
- Selecciona un paciente para ver su historia clinica.
- Archivos: subir radiografias/panoramicas desde la ficha.

## Settings
- Horarios y zona horaria.
- Bloqueos de fechas.
- Turnos (duracion, maximos, sobreturnos).
- WhatsApp y confirmacion.
- Idioma (ES/EN).

## Troubleshooting
- Si no carga, verificar backend en /api/health.
- Si no aparecen horas, revisar tratamiento y fecha.
